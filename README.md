# Face Recognition Attendance System

A simple Python desktop app that uses OpenCV and Tkinter to:

- register a user's face with a user ID and name
- train a face recognition model from saved images
- recognize faces from the webcam
- record attendance in a CSV file

## Features

- face registration with webcam capture
- LBPH face recognition using OpenCV
- attendance logging to `attendance.csv`
- basic Tkinter GUI
- runtime checks for missing camera, missing model file, and missing OpenCV face module

## Requirements

- Python 3
- webcam
- `opencv-contrib-python`
- `numpy`
- `Pillow`

## Install

```powershell
pip install -r requirements.txt
```

## Run

```powershell
python python.py
```

## How To Use

1. Click `Register Face`
2. Enter a new user ID and name
3. Let the app capture face samples
4. Click `Train Model`
5. Click `Recognize Face`

## Project Files

- `python.py` - main application
- `dataset/` - captured face images generated by the app
- `trainer/` - trained model generated by the app
- `attendance.csv` - generated attendance log
- `users.csv` - generated user list

## Notes

- `dataset/`, `trainer/`, `attendance.csv`, and `users.csv` are ignored by Git so generated data and personal records are not uploaded by default.
- If `cv2.face` is missing, install `opencv-contrib-python` instead of plain `opencv-python`.
