## Interview Version

This project was a desktop attendance app that uses computer vision instead of manual entry. I used OpenCV for face detection and LBPH recognition, Tkinter for the interface, and CSV files for lightweight storage. The main challenges were handling webcam access, data collection flow, duplicate registration checks, and making the project clean enough to publish as a portfolio repository.
